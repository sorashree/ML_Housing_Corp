# California Housing Price Prediction 🏠📊

An end-to-end Machine Learning project built while learning the fundamentals of Machine Learning and Scikit-Learn.

## 📌 About the Project

This project uses the California Housing dataset to build a model that predicts the median house value of California districts based on features such as:

* Median income
* Total rooms
* Total bedrooms
* Population
* Households
* Housing median age
* Geographic location
* Ocean proximity

I worked through the complete Machine Learning workflow, from exploring and cleaning the data to preprocessing, training models, evaluating performance, and tuning hyperparameters.

## 🧠 What I Learned

Through this project, I practiced:

* Numerical and categorical feature preprocessing

* Feature engineering
* Creating custom Scikit-Learn transformers
